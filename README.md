# ML Math Image Processing (C#)

Bu proje, C# ve Windows Forms kullanılarak geliştirilen basit bir görüntü işleme uygulamasıdır.  
Proje kapsamında matematiksel görüntü işleme yöntemleri uygulanmıştır.

## Kullanılan Teknolojiler
- C#
- Windows Forms (.NET Framework)
- Visual Studio

## Uygulama Özellikleri
- Bilgisayardan resim seçme
- Gri tonlama (Grayscale) işlemi
- Histogram eşitleme ile kontrast artırma
- İşlenen resmi kaydetme

## Nasıl Çalışır?
1. Resim Seç butonu ile bilgisayardan bir görsel seçilir
2. Gri Tonlama butonu ile görüntü siyah-beyaz hale getirilir
3. Histogram Eşitle butonu ile görüntünün kontrastı artırılır
4. Kaydet butonu ile işlenen görüntü bilgisayara kaydedilir

## Proje Amacı
Bu projenin amacı, pikseller üzerinde matematiksel işlemler yaparak temel görüntü işleme mantığını anlamaktır.  
Histogram eşitleme işlemi, kümülatif dağılım fonksiyonu (CDF) kullanılarak uygulanmıştır.

## Geliştirilebilir Özellikler
- Threshold (siyah-beyaz dönüşüm)
- Negatif görüntü
- Parlaklık ve kontrast ayarları
- Histogram grafik gösterimi
